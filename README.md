This repository contains the source code for a streamlit dashboard on application of one-period binomial model on popular US stocks. 
The data is fetched through yahoo finance.
The model allows you to select your stock's different type of options(call and put), for different expiry dates and different volatility measures of trading windows, and on the basis of that, outisde of a threshhold range, it allows you generate buy/hold/sell signals for the option.
you can also select your desired risk-free rate and the threshhold range.
